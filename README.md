# abolfazlsajadi.com — Personal CV & Portfolio

Static personal website for **Abolfazl Sajadi** — Digital Hardware Engineer · Hardware Security
(PhD candidate, LIACS, Leiden University).

- Single-page portfolio (`index.html`) plus thesis detail pages (`thesis-*.html`).
- Vanilla HTML / CSS / JS, no build step, fully self-contained (no external runtime dependencies).
- Hardware/circuit visual theme; light & dark modes; responsive; accessible.
- Deployed with GitHub Pages to the custom domain **abolfazlsajadi.com**.

## Local preview
```bash
python3 -m http.server 8080
# then open http://localhost:8080
```
